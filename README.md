# AnyApiToolBox
目前，本软件接入了 小歪(api.ixiaowai.cn) 和 搏天(api.btstu.cn) 两大 API，使用到了各 API 的部分功能。  
本软件的核心功能为「批量」，包括但不限于批量下载、批量输出+保存。  
当然，您也可以自行输入一个 URL 来使用功能。
